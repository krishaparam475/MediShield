# MediShield
🚑 MediShield – Emergency Health Card System
A privacy-first emergency health platform that gives first responders instant, secure access to life-saving patient data — while keeping it completely hidden from random bystanders.
🔐 AES-256 encrypted records · 🩺 Role-based dashboards for Patients, Hospitals, Ambulances & Admins · 📍 Geo-tagged scan logs with real-time alerts · 📴 Offline mode for ambulances in dead zones · 🩸 One-tap blood donation toggle
Built as a hackathon MVP with a clear roadmap toward national EMR integration, AI-assisted triage, and wearable/IoT sync.
